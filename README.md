Objective: 
Developed a Python-based customer support ticket analytics system using Pandas and data visualization libraries, 
automating ticket categorization, analyzing customer issue trends, and generating KPI reports to improve operational decision-making.

Key Factors:
Developed an automated ticket analysis system using Python to process and categorize customer support tickets.
Performed data cleaning, preprocessing, and exploratory data analysis (EDA) on customer service datasets.
Identified common customer issues and trends through keyword and sentiment analysis.
Generated actionable insights by analyzing ticket volume, resolution time, and issue categories.
Created interactive visualizations and reports to support business decision-making.
Automated data processing workflows, reducing manual analysis effort and improving reporting efficiency.
Utilized Pandas and NumPy for large-scale data manipulation and statistical analysis.
Built dashboards/charts to track key performance indicators (KPIs) such as response time and ticket resolution rates.
Improved customer support monitoring by providing data-driven recommendations.
